# AI-Powered Phishing & Fraud Detection Platform
## Overview
Web application that analyzes URLs for phishing risk with secure authentication.
## Features
- SHA-256 hashed password storage
- Parameterized SQL queries (SQLi safe)
- URL risk scoring engine
- Dashboard interface
- Report logging
## Tech Stack
- Python Flask
- SQLite
- HTML/CSS
- Cybersecurity concepts
## How to Run
pip install flask
python app.py
Open http://127.0.0.1:5000
